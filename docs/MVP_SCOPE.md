# MVP Scope

Last updated: 2026-06-28

## Must Ship

- Mobile-first Nonogram board.
- 5x5 starter puzzles and 8x8 follow-up puzzles.
- Fill, mark, clear, undo, and solved detection.
- Auto-generated row and column clues.
- Local progress save.
- Pantry Album for completed puzzles.
- Pip reaction moments.
- Visible version label.
- Basic settings: sound toggle and reset progress.

## Week-One Android Target

- Web MVP runs locally and builds cleanly.
- Capacitor Android project can be generated.
- Android app ID reserved as `com.sunnyspoonstudios.pipspicturepantry`.
- Store-facing title, short description, app icon candidate, and screenshots prepared.
- Signed AAB path documented before internal testing.

## iOS Preparation Target

- Bundle ID reserved as `com.sunnyspoonstudios.pipspicturepantry`.
- Web app shell avoids Android-only assumptions.
- Icons and launch metadata kept platform-neutral.
- App Store metadata draft prepared for Mac Mini packaging.

## Trial Monetization

- Most puzzle content remains free.
- Monetization should be optional and light.
- First candidate: one low-price support purchase or small bonus puzzle pack.
- No currency, gacha, aggressive timers, forced ads, or pressure mechanics in MVP.

## Out Of Scope

- Login.
- Cloud save.
- Parent Corner.
- Education settings.
- Restaurant management.
- Village map.
- Shop/currency loops.
- Social sharing.
- Heavy story cards.
